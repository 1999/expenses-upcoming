## ADDED Requirements

### Requirement: navigate-prev-month

The user must be able to move to the previous month easily from the current view.

#### Scenario: click-left-arrow

- **WHEN** the user is viewing `/2026/01` and clicks the left arrow in the NavBar
- **THEN** they are navigated to `/2025/12`

### Requirement: navigate-next-month

The user must be able to move to the next month easily from the current view.

#### Scenario: click-right-arrow

- **WHEN** the user is viewing `/2026/01` and clicks the right arrow in the NavBar
- **THEN** they are navigated to `/2026/02`
